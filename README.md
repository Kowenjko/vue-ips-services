# ServieHub

Команда для запуска `backend` - `localhost:3000`

```bash
docker compose up -d
```

### Технический бэклог

1. В `useUserStore` необходимо реализовать `assert function` для ошибки
2. На странице `LoginView` инкапсулировать логику авторизации
3. Написать отдельный обработчик для route guards
4. Убрать инициализацию токена из `App.vue`
